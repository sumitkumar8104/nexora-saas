# Nexora — Modern SaaS Platform

A modern, conversion-focused SaaS marketing platform built with **Next.js 14** and **Tailwind CSS**. Features an animated gradient hero, interactive workflow showcases, a spotlight feature grid, customer testimonials, and a complete authentication flow.

🔗 **Live Demo:** https://nexora-saas-sk.netlify.app

## ✨ Features
- Animated gradient hero with motion-on-scroll reveals
- Spotlight feature grid & interactive workflow cards
- Testimonials carousel and pricing/CTA sections
- Full auth UI — sign in, register, reset password
- Fully responsive, dark premium aesthetic
- Optimized images and fast first load

## 🛠️ Tech Stack
**Next.js 14** · React 18 · TypeScript · Tailwind CSS · Framer Motion

## 🚀 Getting Started
```bash
npm install
npm run dev      # http://localhost:3000
```

## 📦 Build
```bash
npm run build
npm start
```
